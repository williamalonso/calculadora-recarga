🤔 Sobre o Projeto?

Este é um site simples para calcular o custo de recarga de um veículo elétrico (baseado em uma bateria de 44,9 kWh).

O usuário pode inserir o preço do kWh e a porcentagem atual da bateria para ver o custo de uma carga completa (0-100%) e o custo para completar a carga (da % atual até 100%). O projeto foi focado em uma interface limpa (UI) e uma boa experiência de usuário (UX), com cálculos em tempo real.

🚀 Tecnologias

Esse projeto utiliza as seguintes tecnologias:

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
- [Tailwind](https://tailwindcss.com/)

🙅 Instalação e Uso

Este projeto consiste em um único arquivo HTML e não requer instalação de pacotes ou um ambiente Node.js.

Opção 1: Download Direto

Baixe o arquivo calculadora_recarga.html.

Abra-o diretamente em qualquer navegador web (Chrome, Firefox, Safari, etc.).

Opção 2: Clonando o Repositório

Caso queira clonar o repositório:

```
# Clone o repositório
$ git clone git@github.com:williamalonso/calculadora-recarga.git
```

Após clonar, entre na pasta e simplesmente abra o arquivo index.html no seu navegador. Não é necessário rodar npm install ou npm run dev.
